# TarixiTimeline — MVP

Интерактивный навигатор по истории: таймлайн, фильтры, поиск, мини-карта и пользовательские коллекции. MVP-версия работает в браузере без сборки (no-build).

## Требования

- Современный браузер с поддержкой ES6 (Chrome, Firefox, Edge, Safari)
- Подключение к интернету (подгружаются React, Babel и шрифты из CDN)
- (Опционально) Любой локальный HTTP-сервер для стабильного запуска

## Быстрый старт

### Способ 1. Открыть напрямую (для быстрой проверки)

Откройте файл `index.html` двойным кликом в браузере.

> **Важно:** при открытии через `file://` возможны ограничения CORS в некоторых браузерах. Если интерфейс не загружается или в консоли (F12) ошибки — используйте Способ 2.

### Способ 2. Через локальный сервер (рекомендуется)

**VS Code + Live Server**
1. Установите расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. Откройте проект в VS Code
3. Правый клик на `index.html` → **Open with Live Server**

**Node.js (npx)**
```bash
npx serve .
# или
npx http-server . -p 3000
```

**Python**
```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```
Затем откройте http://localhost:3000

**PHP**
```bash
php -S localhost:3000
```

## Структура проекта

```
├── index.html               # Точка входа, загружает все скрипты и стили
├── styles.css               # Глобальные стили и CSS-переменные
├── data.jsx                 # Данные: фасеты, теги, объекты таймлайна
├── app.jsx                  # Корневой React-компонент, состояние приложения
├── timeline.jsx             # Компонент таймлайна (канвас/рендеринг ленты)
├── detail.jsx               # Панель детальной информации по объекту
├── legend.jsx               # Боковая панель фильтров (теги, типы)
├── tweaks-panel.jsx         # Плавающая панель настроек (вайб, плотность, шкала)
├── universe-manager.jsx     # Управление коллекциями (вселеннами)
├── test-*.html              # Тестовые и концепт-страницы
└── *.md                     # Документация по проекту
```

## Что умеет MVP

- **Таймлайн** — визуальная лента событий, личностей и эпох от -42000 до 2500 года
- **Фильтрация** — по тегам (область + место), типам объектов и подтипам
- **Поиск** — мгновенный поиск по названию в шапке
- **Мини-карта** — обзор всей шкалы с текущим окном просмотра
- **Коллекции (Universe)** — создание, редактирование и переключение между наборами данных
- **Добавление объектов** — форма создания новых событий, личностей и периодов прямо в интерфейсе
- **Пользовательские теги** — создание своих тегов с цветами
- **Настройки отображения** — три вайба (Данные / Архив / Космос), три плотности, три режима шкалы
- **Маркер настоящего** — визуальный буфер "настоящего времени" на шкале

## Настройка по умолчанию

В `index.html` в блоке `TWEAK_DEFAULTS` можно задать стартовые параметры:

```js
const TWEAK_DEFAULTS = {
  "theme": "dark",          // тема (пока только dark)
  "vibe": "data",           // визуальный стиль: data | archive | constellation
  "density": "normal",      // плотность: compact | normal | spacious
  "showMinimap": true,      // показывать мини-карту
  "showWorld": true,
  "showConnections": false, // связи между объектами
  "presentBuffer": true     // маркер настоящего времени
};
```

Измените значения и перезагрузите страницу — настройки применятся автоматически.

## Технологии

- **React 18** (UMD, загружается из unpkg)
- **Babel Standalone** — транспиляция JSX прямо в браузере
- **Vanilla CSS** — кастомные свойства, без препроцессоров
- **Google Fonts** — шрифты Manrope, Fragment Mono, Instrument Serif
- **No build step** — никаких npm install, Webpack или Vite не требуется

## Возможные проблемы

| Проблема | Решение |
|---|---|
| Белый экран / не загружается | Откройте консоль (F12 → Console). Убедитесь, что браузер не блокирует CDN-скрипты (React/Babel). Попробуйте запустить через локальный сервер |
| Ошибка CORS | Запускайте только через `http://localhost`, не через `file://` |
| Изменения в JSX не применяются | Babel кэширует результаты. Сделайте жёсткую перезагрузку: **Ctrl+Shift+R** или **Ctrl+F5** |
| Шрифты не загружаются | Проверьте доступ к `fonts.googleapis.com` (возможно, нужен VPN) |
